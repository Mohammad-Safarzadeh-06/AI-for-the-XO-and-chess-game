This project is currently for the game XO. In the future, I am going to implement the same machine learning model for chess. This model uses statistics and probability to choose its moves based on past games. However, since I have not played much with it so far, it does not know how to play correctly. If I had time, I would like to expand its analysis on games and make it stronger. I would even make the game graphical. I wrote this model on my own and did not receive any help from artificial intelligence, so it does not have a very standard structure. To the extent that I use a txt file to save its past games. One of the problems I encountered was that when someone plays unintelligently with this program, unfortunately, their unintelligent play is also recorded in the learning process, which is one of the challenges I am trying to solve.
